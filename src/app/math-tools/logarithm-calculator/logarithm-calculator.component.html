What is a Logarithm?

A logarithm is the power to which a number (the base) must be raised to produce a given number. For example, log2(8) = 3 because 23 = 8.

This calculator allows you to compute the logarithm of any number to any base. Simply enter the number and the base to see the result.

Formula: logbase(number) = log(number) / log(base)

Example: log10(100) = 2 because 102 = 100.
